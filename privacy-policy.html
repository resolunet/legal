Politique de confidentialité — Fiduo

Dernière mise à jour : 1er mai 2026

Fiduo est une application mobile éditée par Resolunet (ci-après « nous », « notre » ou « l'éditeur »).

Cette politique décrit comment Fiduo traite — ou plutôt ne traite pas — vos données personnelles.

1. Principe fondamental : aucune donnée ne quitte votre appareil

Fiduo est conçue comme une application 100 % locale. Toutes les informations que vous saisissez (dépenses, charges récurrentes, dettes, catégories, paramètres, seuil d'alerte, thème) sont stockées exclusivement sur votre appareil, dans une base de données chiffrée locale (SQLCipher, AES-256).

Concrètement :

  • Aucun compte utilisateur n'est créé.
  • Aucune connexion réseau n'est établie par l'application pour transmettre vos données.
  • Aucun serveur ne reçoit, ne stocke, ni ne traite vos données financières.
  • Aucun tiers (analytics, publicité, sociétés affiliées) n'a accès à vos saisies.

3. Données collectées par Apple ou Google (hors de notre contrôle)

Lorsque vous installez Fiduo via l'App Store (Apple) ou le Google Play Store, ces plateformes peuvent collecter, indépendamment de l'éditeur :

  • Des données de crash et de performance (si vous avez accepté de les partager avec les développeurs dans les paramètres de votre appareil).
  • Des statistiques d'installation et d'engagement anonymisées et agrégées, selon leurs propres politiques de confidentialité.

Ces données sont régies par :

  • Politique de confidentialité d'Apple
  • Politique de confidentialité de Google

Resolunet ne contrôle ni ne reçoit ces données dans une forme identifiant un utilisateur individuel.

4. Suppression des données

Vos données sont stockées localement. Vous pouvez les supprimer à tout moment, de deux manières :

  • Réinitialisation depuis Fiduo : Paramètres → Données → « Réinitialiser les données ». Cette action efface toutes vos dépenses, charges récurrentes, dettes et remboursements. Les catégories et paramètres sont conservés.
  • Désinstallation de l'application : supprime l'intégralité de la base de données locale, y compris les catégories et paramètres.

Aucune action de notre part n'est requise puisqu'aucune copie n'existe en dehors de votre appareil.

5. Sécurité

Les données sont chiffrées au repos sur votre appareil via SQLCipher (AES-256). La sécurité physique de l'appareil (verrouillage par code, biométrie, chiffrement système) reste sous votre responsabilité.

Si votre appareil est perdu ou volé, vos données restent protégées par le chiffrement local mais peuvent être accessibles à toute personne disposant d'un accès physique non protégé à votre appareil.

6. Sauvegarde

Fiduo n'effectue aucune sauvegarde automatique dans le cloud à ce stade. Si vous changez d'appareil ou désinstallez l'application sans précaution, vos données seront définitivement perdues.

Une fonctionnalité de sauvegarde et de restauration est prévue dans une future version.

7. Vos droits (RGPD)

Bien que Fiduo ne nous transmette aucune donnée vous concernant, le Règlement Général sur la Protection des Données (UE 2016/679) vous garantit les droits suivants :

  • Droit d'accès, de rectification, d'effacement
  • Droit à la limitation et à l'opposition au traitement
  • Droit à la portabilité

Ces droits s'exercent directement dans l'application : vos données sont sous votre contrôle exclusif sur votre appareil. Aucune demande RGPD adressée à Resolunet ne pourrait être satisfaite, faute de données à traiter de notre côté.

8. Modifications de cette politique

Cette politique peut être mise à jour pour refléter les évolutions de Fiduo (par exemple, l'ajout d'une fonctionnalité de sauvegarde cloud ou de notifications push). Toute modification matérielle sera notifiée dans l'application avant sa mise en œuvre.
